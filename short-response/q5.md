# Question

With React Router, which of the following path(s) will match the `/products/:id` path?

1. `/products`
2. `/products/apple`
3. `/products/5`
4. `/products/5/about`

Explain your answer.

# Answer

3 and 4 since both of them have an `"id:"` in it with the number 5 being there.
